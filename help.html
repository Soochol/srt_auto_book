SRT Auto Booker v2.0 사용 설명서

소개

SRT Auto Booker v2.0은 SRT 예매 사이트에서 체크박스 방식으로 원하는 시간대와 좌석 등급을 선택하여 자동으로 예약해주는 크롬 확장 프로그램입니다.

🆕 v2.0 새로운 기능:

  • 🎯 시간대별 일반실/특실 체크박스로 정확한 선택
  • 🎵 예약 성공시 축하 멜로디 재생
  • 💾 체크박스 선택 상태 자동 저장
  • 🔄 매진→예약가능 상태 변화 자동 감지

설치 방법

  1. 이 저장소(폴더)를 압축 해제합니다.
  2. 크롬 주소창에 chrome://extensions/ 입력 후 이동합니다. (클릭하면 복사됩니다)
  3. 오른쪽 상단의 개발자 모드를 켭니다.
  4. "압축해제된 확장 프로그램을 로드" 버튼을 클릭하고, 압축을 푼 폴더를 선택합니다.
  5. 확장 프로그램이 목록에 추가되면 설치 완료!

📋 사용 방법 (v2.0)

  1. SRT 예매 사이트 (https://etk.srail.kr)에 접속합니다. (클릭하면 복사됩니다)
  2. 출발지, 목적지, 날짜를 설정하고 조회하기를 클릭해서 열차 시간표를 확인합니다.
  3. 각 열차의 일반실/특실 칸에 "일반"/"특실" 체크박스가 자동으로 나타납니다.
  4. 원하는 시간대와 좌석 등급의 체크박스를 선택합니다.
    • 🟦 "일반" = 일반실 예약
    • 🟨 "특실" = 특실 예약
    • 매진 상태에서도 체크박스 선택 가능
  5. 크롬 우측 상단의 확장 프로그램 아이콘(🚂)을 클릭해 팝업을 엽니다.
  6. 새로고침 간격(ms)을 설정합니다. (기본값: 3000ms = 3초)
  7. "모니터링 시작(F4)" 버튼을 클릭하거나 ⌨️ F4 키를 누릅니다.
  8. 체크박스가 선택된 열차가 예약 가능해지면 자동으로 예약 버튼을 클릭합니다!
  9. 예약 성공 시 🎵 축하 멜로디가 재생되고 모니터링이 자동 중지됩니다.
  10. "중지 (ESC)" 버튼 또는 ESC 키로 언제든 중지할 수 있습니다.

🎯 주요 기능 (v2.0)

  • 🎯 정확한 선택: 시간대별 일반실/특실 체크박스로 원하는 좌석만 예약
  • 💾 자동 저장: 체크박스 선택 상태가 자동으로 저장되어 새로고침해도 유지
  • 🔄 실시간 감지: 매진→예약가능 상태 변화를 실시간으로 감지
  • 🎵 축하 소리: 예약 성공시 도-미-솔-도 멜로디 재생
  • ⚙️ 간격 설정: 새로고침 간격 조절 가능 (설정값 자동 저장)
  • ⌨️ 키보드 단축키: F4로 시작, ESC로 중지
  • 📢 알림 시스템: 예약 성공/실패, 입력 감지 등 상황별 알림
  • 🔍 입력 감지: 사용자가 폼을 입력하는 중에는 자동으로 모니터링 일시 중지

⌨️ 키보드 단축키

  • F4 키: 모니터링 시작 (체크박스 선택 필수)
  • ESC 키: 모니터링 중지

※ 키보드 단축키는 SRT 조회페이지에서만 작동합니다.

⚠️ 주의사항

  • 체크박스 선택 필수: 원하는 열차의 체크박스를 반드시 선택해야 자동 예약됩니다.
  • 사이트 구조 변경: SRT 예매 사이트 구조가 변경되면 동작이 달라질 수 있습니다.
  • 입력 중 일시정지: 폼 입력 중에는 모니터링이 자동으로 일시 중지됩니다.
  • 브라우저 팝업 제한: 브라우저 네이티브 팝업(양식 다시 제출 등)은 자동으로 닫을 수 없습니다.
  • 개인 사용 목적: 본 확장 프로그램은 개인용 자동화 도구로, 상업적/불법적 사용을 금합니다.
  • 예약 확인: 자동 예약 후 반드시 예약 내역을 확인하시기 바랍니다.
